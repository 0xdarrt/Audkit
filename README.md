# Audkit: Sovereign Financial Clarity for the Modern Family

Audkit is a premium financial SaaS platform designed to bring order to the fragmented wealth of middle-class families. We focus on the high-fidelity tracking of traditional Indian assets like Fixed Deposits, LIC Policies, and Gold Bonds, unified under a single, secure vault.

## 🚀 Features

- **3D Interactive Dashboard**: Visual class-based asset tracking.
- **Family Hub Synergy**: Split and sync wealth across generations with custom ownership assignments.
- **Deep Cashflow Intelligence**: An algorithmic engine that computes savings rates and flags high-variable spending.
- **Enterprise Notification Engine**: Professional maturity alerts months in advance to prevent "Frozen Capital."
- **SaaS Monetization ready**: Built-in freemium gating and premium upsell flows.

## 🛠️ Tech Stack

- **Frontend**: React + Vite + Tailwind (Glassmorphism design system)
- **Database**: Supabase (PostgreSQL)
- **Icons**: Lucide-React
- **Typography**: Sora & DM Serif Display

## 📦 Setup & Installation

1.  **Clone & Install**:
    ```bash
    npm install
    ```
2.  **Environment Sync**:
    Create a `.env` file with your `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## 🔒 Security

Audkit utilizes Supabase Row-Level Security (RLS) to ensure that your financial data is isolated and encrypted. Family synchronization is managed through secure junction tables that respect modern privacy standards.
